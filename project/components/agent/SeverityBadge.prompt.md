# SeverityBadge

Severity chip for critique items. Fixed color mapping: `blocker` red, `major` amber, `minor` cyan, `info` muted.

```jsx
<SeverityBadge level="blocker" />
<SeverityBadge level="major" />
<SeverityBadge level="minor" />
```

Pairs with `CritiqueCard`; also used as the badge count source on the "Thinking & Critiques" trigger.
